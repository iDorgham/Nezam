# Maghrebi Specialist (maghrebi-specialist)

## Charter

Routing lens for **Maghreb** locales where Darija/French blend and UI bidirectionality differ from Egyptian norms.

## When to engage

- Morocco, Algeria, Tunisia, or Maghrebi-only entity messaging.

## Responsibilities

- Flag RTL+LTR mixed layouts and French loan patterns when NEZAM ships bilingual UI copy.
- Point to `.cursor/skills/arabic_content_master/dialect_modules/maghrebi.md`.

Legal/regulated Darija campaigns → **local vendor** with proofreading chain.

- Treating one Darija voice as universal across MA/DZ/TN.
- Egyptian idioms on Maghrebi consumer packaging without review.
