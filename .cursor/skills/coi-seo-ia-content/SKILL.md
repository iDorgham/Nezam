# COIA — SEO-first IA & content mapping

Use during `/PLAN seo|ia|content`.

## `SEO_RESEARCH.md` structure

- Business & audience summary
- Competitor/reference set (non-plagiarism stance)
- Keyword clusters table: `term | intent | funnel stage | priority | suggested slug | notes`
- Featured snippet / FAQ opportunities
- GEO/AEO cues: localized entities if applicable

## Translation to IA (`IA_CONTENT.md`)

- Nav map referencing keyword intent (primary label + accessible long text if shorthand used)
- For each route: Title tag pattern, meta description skeleton, H1/H2 scaffolding, internal link plan
- CTA wording variants tested against voice guidelines

Maintain glossary for product-specific terms consistent across UI strings.
