# Skill: NEZAM Design Server

This skill allows agents to interact with the NEZAM Design Server to manage sitemaps, wireframes, and design tokens.

## Commands
- `npm run design-server` — Start the server (from repo root).
- `curl http://localhost:4000/api/context` — Read current context.

## Usage
Use this skill when you need to lock the design contract or modify the sitemap.
