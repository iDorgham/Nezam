# Workspace — Claude Code handoff

**Project:** {{PROJECT_NAME}}

## Open Claude Code

1. Open this repository as the project/workspace in your editor where **Claude Code** is installed.
2. Optional: add repo-level guidance with **`/CREATE claude-md`** in Cursor (creates root `CLAUDE.md` from template when missing).
3. Start a new Claude Code session scoped to this repo.

## Paste the mission below into Claude Code

Use the **same mission block** as in **`docs/core/required/CLAUDE_CLI_PLAN.md`**: read **Before you write anything new**, then execute **Mission** (strict planning order) and **Completion**.

If that file is missing in your clone, regenerate it with **`/CREATE claude-cli-prompt`** in Cursor.

## After Claude Code finishes

Return to Cursor; run **`/PLAN ...`** to refine or **`/GUIDE next`** for the safe next command.
