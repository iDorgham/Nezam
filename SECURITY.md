# Security Policy

## Supported Versions

Only the latest version of NEZAM is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | ✅ Yes              |
| < v1.0  | ❌ No               |

## Reporting a Vulnerability

Please do not report security vulnerabilities via public GitHub issues. 

Instead, please report vulnerabilities by contacting the maintainers directly at security@iDorgham.com.

We will acknowledge receipt of your report within 48 hours and provide a detailed response within 7 days, including a plan for remediation and disclosure.
