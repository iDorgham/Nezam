# Documentation Index

Documentation is organized by category:

- `docs/context/` — runtime/project memory and workspace references.
- `docs/specs/` — PRD, SDD, and feature specifications.
- `docs/prompts/` — prompt documents and assistant handoff prompts.
- `docs/templates/` — reusable authoring templates.
- `docs/reference/` — canonical references, external kits, and rules mirrors.
- `docs/reports/` — progress and reporting artifacts.
- `docs/external-ai/` — external assistant operating guidance.

Top-level markdown files are intentionally minimized; reference catalogs live in category folders (for example, `docs/reference/external-kits/CATALOG.md`).
