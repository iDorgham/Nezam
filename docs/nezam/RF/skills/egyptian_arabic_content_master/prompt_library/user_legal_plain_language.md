# User template — Legal-adjacent plain language (UX / marketing)

## Placeholders

`{{surface}}` `{{topic}}` `{{audience}}` `{{approved_facts}}` `{{must_link_to}}`

## Template

"اكتب نصاً عربياً (مصرية واضحة مع فصحى للعناوين إن لزم) لـ {{surface}} حول {{topic}}. الجمهور: {{audience}}.  
حقائق معتمدة من العميل فقط: {{approved_facts}}.  
إحالة إلزامية للوثيقة الكاملة: {{must_link_to}}.  
أضف تنويهاً أن المحتوى ليس استشارة قانونية. لا تخترع بنوداً ملزمة."

## Example

"مربع موجز أعلى صفحة سياسة خصوصية لتطبيق توصيل طعام. جمهور: مستخدمون مصريون. حقائق معتمدة: 'لا نبيع بياناتك لطرف ثالث للتسويق؛ نحتفظ بالعنوان 12 شهراً بعد آخر طلب'. رابط: /privacy. تنويه: ليس استشارة قانونية."
