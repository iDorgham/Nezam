# User template — Video script (Reels / Shorts / ad)

## Placeholders

`{{duration_sec}}` `{{product}}` `{{tone}}` `{{platform}}` `{{hook_style}}` `{{cta}}` `{{banned_claims}}`

## Template

"سكريبت فيديو {{duration_sec}} ثانية لـ {{product}}، منصة {{platform}}، نبرة {{tone}}. أسلوب الافتتاحية: {{hook_style}}. CTA: {{cta}}.  
تجنب تماماً: {{banned_claims}}.  
سلّم: نص VO جاهز + supers منفصلة + إشارة موسيقى/إيقاع مقترح."

## Example

"45 ثانية، منتجع في مرسى علم، إنستغرام ريلز، **Storytelling + Friendly**، افتتاحية صوت موج، CTA: حجز عبر الموقع. ممنوع: ذكر أسعار محددة أو ضمان نتائج طبية."
