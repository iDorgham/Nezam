# Content Map — Nightclub System (Egypt Focus)

> **Focus:** Voice, tone, and key copy for Egyptian users (mix of Masri and English).

## 1. Voice & Tone

### 1.1 Public / Guest Facing
- **Tone:** Energetic, upscale, and exclusive.
- **Language:** Egyptian Arabic (Masri) for a friendly, local feel, and English for the premium vibe.
- **Style:** Concise and action-oriented.

### 1.2 Internal Apps (Sales / Security / Admin)
- **Tone:** Professional, direct, and efficient.
- **Language:** Formal Arabic or English (depending on user preference).
- **Style:** Minimal text, clear labels, and high scannability.

## 2. Key Page Copy (Drafts)

### 2.1 Landing Page / Discovery
- **Hero Title (Arabic):** احجز سهرتك في ثواني
- **Hero Title (English):** Book Your Night in Seconds
- **Subtitle (Arabic):** طاولات VIP وجيست ليست لأحسن أماكن السهر في مصر.
- **Subtitle (English):** VIP tables and guest lists for the best venues in Egypt.

### 2.2 Table Booking Page
- **Title (Arabic):** اختر ترابيزتك
- **Title (English):** Choose Your Table
- **Instructions (Arabic):** اضغط على الترابيزة المتاحة لتحديدها وإتمام الحجز.

### 2.3 Guest Digital Pass (QR Page)
- **Welcome (Arabic):** منورنا! جاهز للسهرة؟
- **Welcome (English):** Welcome! Ready for the night?
- **Instructions (Arabic):** وري الكود ده للأمن عند الباب عشان تدخل علطول.
- **Instructions (English):** Show this code to security at the door for fast entry.
