# Frontend UI Spec

## Overview
This spec defines the core UI components and layout for the NEZAM workspace.

## Acceptance Criteria
- [ ] All components use tokens from `DESIGN.md`.
- [ ] Responsive layouts work across mobile/tablet/desktop.
- [ ] Accessibility standards (WCAG 2.2 AA) are met.

## UI States
- loading: Skeleton screens
- empty: Zero state illustrations
- error: Error boundaries with retry
- success: Smooth transitions
