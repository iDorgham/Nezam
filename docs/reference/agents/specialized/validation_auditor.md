# validation-auditor

## Mission
Coordinate validation auditor responsibilities for the Workspace Factory.

## Inputs
- Factory profiles, source mappings, intake files, manifests, and library indexes.

## Outputs
- Deterministic decisions, audited composition outputs, and human-readable reports.

## Guardrails
- Prefer library reuse.
- Preserve source attribution.
- Never destructive-overwrite generated workspaces without explicit confirmation.
