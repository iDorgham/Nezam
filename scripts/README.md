# Scripts Index

Scripts are organized by category:

- `scripts/checks/` — repository and CI readiness checks.
- `scripts/context/` — context doc synchronization and hook installers.
- `scripts/release/` — release planning helpers.
- `scripts/testing/` — local validation and smoke tests.
- `scripts/ui/` — shared terminal UI formatting utilities.
